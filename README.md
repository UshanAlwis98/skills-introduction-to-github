## 🌱 GitHub Foundations

> Validate your understanding of the core concepts, products, and collaboration workflows that power GitHub.

**Exam Code:** `GH-900` &nbsp;|&nbsp; **Level:** Beginner &nbsp;|&nbsp; **Duration:** 100 minutes

### 📚 Topics Covered

| Domain                  | Key Concepts                                          |
| :---------------------- | :---------------------------------------------------- |
| **Git & GitHub Basics** | Version control, repositories, Git commands           |
| **Collaboration**       | Pull Requests, Issues, Discussions, code review       |
| **GitHub Products**     | Plans, Codespaces, GitHub Mobile, GitHub CLI          |
| **Project Management**  | Projects, Milestones, Labels, GitHub Insights         |
| **Privacy & Security**  | Permissions, branch protection, 2FA                   |
| **Community**           | Open source contributions, Markdown, GitHub Community |

## 🤖 GitHub Copilot

> Prove your expertise in AI-assisted development and responsible use of GitHub's AI coding tool.

**Exam Code:** `GH-200` &nbsp;|&nbsp; **Level:** Beginner → Intermediate &nbsp;|&nbsp; **Duration:** 100 minutes

### 📚 Topics Covered

| Domain                 | Key Concepts                                              |
| :--------------------- | :-------------------------------------------------------- |
| **AI Fundamentals**    | How Copilot works, responsible AI, limitations            |
| **Prompt Engineering** | Zero-shot vs few-shot, context management                 |
| **Copilot Features**   | Code completion, Copilot Chat, CLI, agent mode            |
| **Plans & Privacy**    | Subscription tiers, data handling, content exclusions     |
| **Enterprise Use**     | Org-level policies, privacy controls, security deployment |
| **SDLC Integration**   | Testing, debugging, legacy code modernization             |

## ⚡ GitHub Actions

> Demonstrate your ability to automate software workflows, build CI/CD pipelines, and manage GitHub runners.

**Exam Code:** `GH-500` &nbsp;|&nbsp; **Level:** Intermediate &nbsp;|&nbsp; **Duration:** 100 minutes

### 📚 Topics Covered

| Domain                     | Key Concepts                                       |
| :------------------------- | :------------------------------------------------- |
| **Workflow Basics**        | YAML syntax, triggers, events, jobs, steps         |
| **Continuous Integration** | Automated testing, build pipelines, matrix builds  |
| **Continuous Deployment**  | Deployment environments, secrets management        |
| **Runners**                | GitHub-hosted vs self-hosted, runner configuration |
| **Actions Ecosystem**      | Marketplace actions, creating custom actions       |
| **Security**               | Permissions, OIDC, secrets, workflow hardening     |

**Exam Code:** `GH-700` &nbsp;|&nbsp; **Level:** Intermediate &nbsp;|&nbsp; **Duration:** 100 minutes

### 📚 Topics Covered

| :------------------------ | :------------------------------------------------- |
| **User Management**       | Teams, roles, SSO, SCIM provisioning               |
| **Repository Governance** | Branch protection, rulesets, CODEOWNERS            |
| **Organization Settings** | Policies, audit logs, enterprise configuration     |
| **Workflow Optimization** | Repository templates, reusable workflows           |
| **Security Practices**    | Permissions model, token management                |
| **Collaboration**         | Efficient code review, project management at scale |

## 🔐 GitHub Advanced Security

> Validate your expertise in securing codebases with GitHub's advanced security toolchain.

**Exam Code:** `GH-800` &nbsp;|&nbsp; **Level:** Advanced &nbsp;|&nbsp; **Duration:** 100 minutes

### 📚 Topics Covered

| Domain                      | Key Concepts                                     |
| :-------------------------- | :----------------------------------------------- |
| **Secret Scanning**         | Detecting exposed credentials, push protection   |
| **Code Scanning**           | CodeQL analysis, SARIF results, custom queries   |
| **Dependency Management**   | Dependabot alerts, security updates, SBOMs       |
| **Vulnerability Detection** | CVEs, severity assessment, remediation workflows |
| **Security Policies**       | Repository security settings, rulesets           |
| **Supply Chain**            | Artifact attestations, signing, provenance       |
